iPhone と Mac で同期する

台帳データをご自身の非公開Gistに保存し、両方の端末で読み書きします。 新しいサービスの登録は要りません。GitHubのトークンを1回発行し、端末ごとに下の欄へ貼り付けてください。

  1. GitHub → Settings → Developer settings → Personal access tokens → Tokens (classic) で新規発行
  2. スコープは gist だけにチェック（他は不要です）
  3. 発行されたトークンを下に貼り付けて「保存して同期」
  4. もう一方の端末でも同じトークンを貼り付ける。保存先のGistは自動で引き継がれます

トークンはこの端末のブラウザ内にのみ保存され、GitHub以外には送信されません。共用の端末では使わないでください。 不要になったら「この端末の同期を解除」でトークンを削除し、GitHub側でも失効させてください。
